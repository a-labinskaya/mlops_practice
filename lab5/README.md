# Lab 5: ML model quality testing

В лабораторной работе создана модель Linear Regression.

Что сделано:

1. Созданы три чистых синтетических датасета.
2. Модель обучена на чистом датасете.
3. Качество модели проверено с помощью pytest.
4. Для оценки используется метрика R2 score.
5. Создан зашумленный датасет.
6. Тесты показывают, что на зашумленных данных качество модели ухудшается.

Файлы:

- lab5.ipynb — основной Jupyter Notebook.
- test_model.py — автоматические тесты pytest.
- model.pkl — сохраненная модель.
- clean_data.png — график чистых данных.
- noised_data.png — график зашумленных данных.
